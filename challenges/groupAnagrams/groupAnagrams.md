## Group anagrams

## Problem

Given an array of strings, group anagrams together.
Output is an array of groups, which are themselves arrays.

## Example

Input: ["eat", "tea", "tan", "ate", "nat", "bat"]

Output:

[

  ["ate", "eat", "tea"],

  ["nat", "tan"],

  ["bat"]

]
