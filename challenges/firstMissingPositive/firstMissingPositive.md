# First missing positive

## Problem
Write a function that takes an unsorted array of integers and returns the
smallest positive integer that is missing from the array.
It should run in O(n) time.

## Examples
```javascript
f([1, 2, 3])
// => 4

f([3, -4, 17, 1])
// => 2

f([99, 100, 101, 102])
// => 1
```
