### Segregate Even and Odd Numbers

Given an array of integers, segregate even and odd numbers in the array. All the even numbers should be present first, and then the odd numbers.

Example:

```
Input : [1, 9, 5, 3, 2, 6, 7]
Output : [2, 6, 5, 3, 1, 9, 7]
```
