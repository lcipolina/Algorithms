# Missing number in array

## Problem

Given an array containing _n_ distinct numbers taken from 0, 1, 2, ..., _n_, find the number that is missing from the array.

## Examples
```
Example 1:

Input: [3,0,1]
Output: 2

Example 2:

Input: [9,6,4,2,3,5,7,0,1]
Output: 8
```
